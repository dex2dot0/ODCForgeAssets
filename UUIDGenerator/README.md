# UUIDGenerator

A simple library that exposes UUID generation to OutSystems.

## Icon Setup

This project now follows the embedded-resource icon pattern required for OutSystems publishing. To rewire or replace the icon from the repository root, run:

```cmd
icon.cmd UUIDGenerator --icon UUIDGenerator\resources\uuid_generator.png
```

## Testing

To run the tests, run `dotnet test UUIDGenerator.Tests/UUIDGenerator.Tests.csproj`.
